TypeRig

Python framework for font design and engineering. Geometric primitives, advanced interpolation with stem compensation, delta scaling, CJK-aware transformations, and FontLab integration through a transparent proxy layer.

Architecture

Core Objects

Pure Python geometry: Point, Line, CubicBezier, Node, Contour, Shape, Layer, Glyph. No external dependencies.

Core Functions

Math, geometry, and transform utilities including adaptive_scale, compensated interpolation, and Hobby splines.

File I/O

XML serialization with the experimental .trfont format — shared glyph pools, manifests, and compact notation.

Proxy Layer

Transparent bidirectional binding to FontLab via __meta__ attribute routing. Eject/inject for round-tripping.

Delta Engine

Directional coordinate system, per-contour stem measurement, and stem-compensated scaling for CJK and Latin alike.

Web Tools

Browser-based Glyph Viewer with Pyodide integration — full TypeRig core running as Python in WebAssembly.

Core Modules
typerig.core.objects Geometric primitives — atom, point, line, cubic/quadratic bezier, node, contour, shape, layer, glyph, anchor, SDF, delta, hobby splines
typerig.core.func Math functions, transform utilities (adaptive scale, compensator, lerp), geometry helpers
typerig.core.fileio XML serialization engine with class registry and compact matrix notation
typerig.proxy.fl FontLab proxy objects — transparent wrappers with attribute routing and lazy loading
Quick Start
# Clone the repository git clone https://github.com/kateliev/TypeRig.git # Core objects — pure Python, no dependencies from typerig.core.objects import Node, Contour, Shape, Layer, Glyph # Transform functions from typerig.core.func.transform import adaptive_scale, lerp # FontLab proxy (inside FontLab only) from typerig.proxy.fl.objects.glyph import eGlyph
